# RMQ问题

> RMQ 是英文 Range Maximum/Minimum Query 的缩写，表示区间最大（最小）值。	——OI wiki

RMQ的常见解法有三种：

1. 单调栈：

   | 预处理时间复杂度 | 询问时间复杂度 | 空间复杂度 |
   | ---------------- | -------------- | ---------- |
   | $O(mlogm)$       | $O(logn)$      | $O(n)$     |

   

2. 线段树：

   | 预处理时间复杂度 | 询问时间复杂度 | 空间复杂度 |
   | ---------------- | -------------- | ---------- |
   | $O(n)$           | $O(logn)$      | $O(n)$     |

   

3. ST表：

   | 预处理时间复杂度 | 询问时间复杂度 | 空间复杂度 |
   | ---------------- | -------------- | ---------- |
   | $O(nlogn)$       | $O(1)$         | $O(nlogn)$ |
