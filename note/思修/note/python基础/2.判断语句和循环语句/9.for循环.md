# for循环

像while循环一样，for可以完成循环的功能。

在Python中 for循环可以遍历任何序列的项目，如一个列表或者一个字符串等。

## for循环的格式

```python
    for 临时变量 in 列表或者字符串等:
        循环满足条件时执行的代码
    else:
        循环不满足条件时执行的代码
```

## demo1

```python
    name = 'dongGe'

    for x in name:
        print(x)
```
