## 三角函数的导数

1. $(\tan x)' = \sec ^2x$
2. $(\cot x)' = -\csc ^2x$
3. $(\sec x)' = \sec x\tan x$
4. $(\tan x)' = -\csc x\tan x$
5. $(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$
6. $(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$
7. $(\arctan x)' = \frac{1}{1+x^2}$
8. $(arccot\ x)' = -\frac{1}{1+x^2}$



## 第二类换元积分法

| 根式             | 三角代换    |
| ---------------- | ----------- |
| $\sqrt{a^2+x^2}$ | $x=a\tan t$ |
| $\sqrt{a^2-x^2}$ | $x=a\sin t$ |
| $\sqrt{x^2-a^2}$ | $x=a\sec t$ |



## 三角函数的积分

**设$u=\tan \frac{x}{2}$**

则有

```math
\sin x = \frac{2u}{1+u^2},\ \cos x = \frac{1-u^2}{1+u^2},\ \tan x = \frac{2u}{1-u^2} \\ dx = \frac{2}{1+u^2}du
```

## 常见高阶导数

$(e^x)^{(n)} = e^x$

$(\sin x)^{(n)}=\sin(x+n\frac{\pi}{2})$

$(\cos x)^{(n)}=\cos(x+n\frac{\pi}{2})$

$[\ln(1+x)]^{(n)}=(-1)^{n-1} \frac{(n-1)!}{(1+n)^n}$
