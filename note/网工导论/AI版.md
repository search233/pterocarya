# 《网络工程导论》2025 复习笔记

## 第4章 计算机网络基础

### 1. 网络应用的用户类型和目标 (P5-7)
- **用户类型**：个人和家庭用户、企业用户、移动用户
- **目标**：资源共享、信息交换、远程访问、协作通信

### 2. 网络发展阶段 (P12)
1. 面向终端的计算机网络
2. 计算机-计算机网络
3. 开放式标准化网络
4. 互联网与高速网络

### 3. 网络的定义 (P15)
计算机网络是指将地理位置不同的具有独立功能的多台计算机及其外部设备，通过通信线路连接起来，在网络操作系统、网络管理软件及网络通信协议的管理和协调下，实现资源共享和信息传递的计算机系统。

### 4. 网络硬件和软件的组成 (P20-31)
- **硬件**：
  - 终端设备：计算机、手机、服务器等
  - 传输介质：双绞线、同轴电缆、光纤等
  - 网络设备：交换机、路由器、网桥等
- **软件**：
  - 网络操作系统
  - 网络协议软件
  - 网络管理软件

### 5. 网络的主要功能及其基本功能
- **主要功能**：资源共享、信息传输、分布式处理、负载均衡
- **基本功能**：数据通信、资源共享

### 6. 光纤特点 (P27)
- 传输速率高
- 带宽大
- 抗电磁干扰
- 传输距离远
- 安全性高

### 7. 网络的拓扑结构 (P33-37)
- 总线型
- 星型
- 环型
- 树型
- 网状型

### 8. 网络的分类-通信技术 (P39-47)
- 广播式网络
- 点对点网络

### 9. 网络的分类-交换技术 (P37-47)
- 电路交换
- 报文交换
- 分组交换

### 10. 网络分层设计的理由/优点 (P48-63)
- 简化设计
- 便于标准化
- 易于维护和升级
- 促进竞争和创新
- 各层独立发展

### 11. 网络的体系结构两种标准
- **ISO/OSI**：七层模型（物理层、数据链路层、网络层、传输层、会话层、表示层、应用层）
- **TCP/IP**：四层模型（网络接口层、网际层、传输层、应用层）

### 12. CSMA/CA 的主要内容 (P71)
- **协议流程**：
  1. 侦听信道（Carrier Sense）
  2. 随机等待时间
  3. 清空信道
  4. 数据发送
  5. 确认和重传
- **作用**：用于局域网、无线局域网等，确保多个设备之间的公平竞争和数据传输的可靠性。

## 第5章 因特网基础

### 13. 网络逻辑地址 (IP) 作用和物理地址 (MAC) 的区别
- **IP地址**：逻辑地址，用于网络层寻址，可变化
- **MAC地址**：物理地址，用于数据链路层寻址，固定

### 14. IP 地址格式
- IPv4：32位，点分十进制表示（如192.168.1.1）
- IPv6：128位，冒号分隔十六进制表示（如2001:0db8:85a3::8a2e:0370:7334）

### 15. 子网掩码定义方式和作用
- **定义方式**：与IP地址相同长度的二进制数，网络部分全1，主机部分全0
- **作用**：划分网络和主机部分，用于子网划分

### 16. 默认网关定义方式和作用
- **定义方式**：本地网络中路由器的IP地址
- **作用**：将数据包转发到其他网络

## 第6章 网站建设、网络信息表示及显示机制

### 17. 网站的设计原则
- 用户友好
- 响应式设计
- 内容清晰
- 导航简单
- 性能优化

### 18. HTML 的各型标记符作用
- 结构标记：`<html>`, `<head>`, `<body>`
- 文本标记：`<p>`, `<h1>`-`<h6>`
- 链接标记：`<a>`
- 图像标记：`<img>`
- 表格标记：`<table>`, `<tr>`, `<td>`

### 19. CSS 插入的三种方式
1. 内联样式：直接在HTML标签中使用`style`属性
2. 内部样式表：在`<head>`中使用`<style>`标签
3. 外部样式表：通过`<link>`标签引入外部CSS文件

### 20. JS 的特点和用途
- **特点**：解释性、动态性、跨平台
- **用途**：网页交互、表单验证、动态内容更新

### 21. HTTP 协议用途
- 用于客户端和服务器之间的通信
- 传输超文本（HTML）和其他资源

## 第7章 语义网基础

### 22. 语义网的发明者和 WEB 发展史
- **发明者**：Tim Berners-Lee
- **发展史**：Web 1.0（静态网页）→ Web 2.0（交互式网页）→ Web 3.0（语义网）

### 23. 知识的表示形式
- **三元组的图形化表示**：主体-谓词-客体
- **文档表示**：RDF/XML, Turtle, JSON-LD

### 24. 知识描述语言的主要标准和区别
1. **RDF**：三元组表示资源，不具备推理能力
2. **RDFS**：表示资源的类型及类型的属性，具有类型推理能力
3. **OWL**：基于RDFS扩展，具有属性推理能力

### 25. 知识查询语言的标准 (P107)
- SPARQL：用于查询RDF数据的查询语言

### 26. 知识图谱建模与序列化
- 建模：使用RDF或OWL构建知识图谱
- 序列化：将知识图谱存储为RDF/XML、Turtle等格式

## 第9章 无线网络与物联网

### 27. 无线电频谱划分与特点
- **划分**：低频、中频、高频、甚高频、超高频
- **特点**：不同频段有不同的传播特性和应用场景

### 28. 无线网络协议层次特点
- 物理层：调制解调
- 数据链路层：帧传输
- 网络层：路由选择

### 29. 无线网络应用目的
- 移动通信
- 无线局域网
- 物联网

### 30. 微波通信特点
- 直线传播
- 带宽大
- 受天气影响

### 31. 多址和复用关系
- **多址**：多个用户共享同一信道
- **复用**：同一信道传输多路信号

### 32. 复用多址技术
- 频分多址（FDMA）
- 时分多址（TDMA）
- 码分多址（CDMA）
